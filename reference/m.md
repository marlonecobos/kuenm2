# SpatVector Representing Calibration Area for *Myrcia hatschbachii*

A spatial vector defining the calibration area used to extract
background points for fitting models of *Myrcia hatschbachii*. The area
was generated by creating a minimum convex polygon around presence
records (`occ_data`), then applying a 300 km buffer.

## Format

A `Spatvector` object.

## Value

No return value. Used with function
[`vect`](https://rspatial.github.io/terra/reference/vect.html) to bring
raster variables to analysis.

## Examples

``` r
m <- terra::vect(system.file("extdata",
                             "m.gpkg",
                              package = "kuenm2"))
terra::plot(m)
```
